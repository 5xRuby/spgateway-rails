# spgateway-rails

An API wrapper for [Spgateway](https://www.spgateway.com).


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spgateway-rails'
```

And then execute:

```bash
$ bundle
```

Finally, run the install generator:

```bash
$ rails generate spgateway:install
```
